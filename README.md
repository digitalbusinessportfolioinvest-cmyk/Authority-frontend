# AuthorityDx Frontend

Vite + React frontend for AuthorityDx.

## Local development

```bash
npm install
npm run dev
```

## Environment variables

Create `.env` from `env.example`:

```bash
VITE_API_URL=http://localhost:3001
```

For production (Netlify), set:

- `VITE_API_URL` = your deployed backend API URL
